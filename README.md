# Jarvis AI Voice Assistant

A voice-activated AI assistant built with Python and ElevenLabs, inspired by Iron Man's Jarvis.

## Demo
> Coming soon

## What it does
- 🎙️ Voice conversation powered by ElevenLabs Conversational AI
- 🔍 Web search via DuckDuckGo
- 💾 Save information to text files
- 🌐 Generate HTML webpages on demand
- 🎨 AI image generation via Pollinations

## Tech Stack
- Python 3.11
- ElevenLabs Conversational AI
- LangChain Community
- OpenAI API
- Pollinations.ai

## Setup
1. Clone the repo
2. Create a virtual environment: `python -m venv .venv`
3. Install dependencies: `pip install -r requirements.txt`
4. Create a `.env` file with your API keys:
ELEVENLABS_API_KEY=your_key
ELEVENLABS_AGENT_ID=your_agent_id
OPENAI_API_KEY=your_key
5. Run: `python main.py`
